
# User Management API

This is a simple API for managing users with CRUD operations (Create, Read, Update, Delete) using Express.js, Joi for validation, and middleware for logging and authentication.

## Features:
- Create new users
- Retrieve user data
- Update existing users
- Delete users
- Authentication & Logging middleware

## How to Run:
1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the server

API is accessible at `http://localhost:5000/api/users`.
